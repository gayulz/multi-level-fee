레이아웃 테스트 페이지

이 페이지는 공통 레이아웃(Header, Sidebar, Footer)이 정상적으로 렌더링되는지 확인하기 위한 테스트 페이지입니다.

검증 항목:

  • 상단 Header 표시 확인 (로고, 프로젝트명, 실시간 시계)
  • 좌측 Sidebar 표시 확인 (메뉴 목록)
  • 하단 Footer 표시 확인 (Copyright 정보)
  • 모바일 화면에서 사이드바 토글 버튼 동작 확인
  • Bootstrap, Font Awesome, jQuery CDN 로드 확인
